# 🚀 AI Career Guide System

🌐 Live Demo: https://ai-career-guide-gnjb.onrender.com

An intelligent AI-powered resume analysis web application that evaluates skills, detects domain alignment, provides career feedback, and generates downloadable reports.

## 🔍 Features
- Resume skill detection
- Domain identification (Technology, Creativity, Business)
- Domain mismatch analysis
- Resume scoring system
- Personalized career feedback
- Action plan suggestions
- PDF report generation

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- PyPDF2 (Resume parsing)
- ReportLab (PDF generation)

## 🎯 How It Works
1. Upload your resume
2. Select your interested domain
3. Get:
   - Resume score
   - Skill analysis
   - Missing skills
   - Career feedback
